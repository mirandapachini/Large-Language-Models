Semantic Similarity

  • Ref
  • Most common: cosine similarity

Example:

A = [0.5, 0.866], B = [0.866, -0.5]

A dot B = 0.5*0.866+0.866*(-0.5) = 0 cos(theta) = 0

A dot B = 0.5*0.5+0.866*0.866 = 0.25+0.75 = 1

||A|| = sqrt(0.5^2 + 0.866^2) = 1 ||B|| = sqrt(0.5^2 + 0.866^2) = 1

cos(theta) = 1/(1*1) = 1
